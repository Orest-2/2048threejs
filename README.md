# 2048

3D 2048 game made with Vue, Three.js and TypeScript

### [Demo](https://wizardly-newton-9d13e7.netlify.app/)

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
